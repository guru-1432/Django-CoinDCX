CoinDCX Report

Why do we need this ?
  • One tool that the coinDCX is missing is place where an inverst/trader can have a clear insight on his portfolio (Some sort of analytics tool as in apps like Zerodha)
  • More stats on transactions like avg unit price of an asset, P/L perecentage etc will be useful for the end user.
This report
  • The main focus of this report is to show some insight of the trades like avg price of a crypto and proft/loss percentage.
  • There is also a chart that shows how our total investment is split across various asset. This will be useful for an inverstor to manage his assets.
Technicality
  • The report is built using Python-Flask and chart.js for the chart.
  • The stats from the report are from the users filled orders.
  • One of the CoinDCX public api was used to get the current market price for a particular asset.
Developer Note
  • This is just a POC to highlight that we can get some usefull insights from the data avilable.
  • Currently the datasource for the application is a flat CSV file extracted from the CoinDCX orders page.
  • As process improvement we can update the datasource to be a database or we can use an existing CoinDCX api.
